# TP Integrador – JavaScript aplicado a una realidad cotidiana

## Tema: Carrito de compras en un kiosco

### Descripción
Este programa simula la compra de productos en un kiosco.  
Permite mostrar los productos disponibles, calcular el total, aplicar descuentos y procesar el pago del cliente.  
Se aplican los principales conceptos de JavaScript de manera sencilla y práctica.

### Contenidos aplicados
- Variables: string, number, boolean, null.  
- Objeto literal con método y uso de `this`.  
- Array recorrido con `forEach()`.  
- Función tradicional y función flecha.  
- Estructuras de control: `if/else`, `switch`, `do...while`.  
- Uso de `bind()` para mantener el contexto del objeto.

### Ejecución
Abrir el archivo `tp.js` en Node.js o en la consola del navegador y ver los resultados.
